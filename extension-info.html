확장 안내는
보조 기능 후보입니다

이 페이지는 고급 모드에서 보조적으로 검토 중인 확장 안내입니다. 핵심 연습 기능은 확장 없이도 사용할 수 있고, 안정화 전에는 선택 기능처럼 안내하지 않습니다.

현재 상태

우선 목표는 링커리어 CBT 결과 페이지의 답지 표를 읽어 정답과 정답률을 고급 모드 OMR에 바로 넘기는 기능입니다. 수동 복사 입력이 먼저 안정화된 뒤 확장 자동화를 붙입니다.

제공 기준

확장은 고급 모드 안에서만 안내합니다. 핵심 연습 도구는 확장 없이도 사용할 수 있어야 하며, 확장은 결과 페이지의 공개 표를 사용자가 직접 열어둔 상태에서만 보조적으로 읽습니다.

출시 전 확인

  • 설치 흐름과 삭제 방법을 짧게 정리합니다.
  • 지원 브라우저와 제한 사항을 확인합니다.
  • 예: cbt.linkareer.com/cbt/practice/result/... 결과 페이지 하단 답지 표에서 NO., 정답, 정답률 열만 추출합니다.
  • 문제 발생 시 책임 범위와 문의 방법을 명확히 표시합니다.
