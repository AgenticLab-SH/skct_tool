자료는 옆에 띄우고,
답안은 여기에서.

문제 PDF나 E-book을 준비하면 됩니다. 이 사이트에서 시간을 재고, 답안을 표시하고, 채점하세요.

연습 도구 열기고급 기능 보기
1. 자료 열기문제 PDF·E-book을 화면 왼쪽에 둡니다.
2. 팝업으로 열기연습 도구를 오른쪽에 두면 마킹이 편합니다.
3. 풀고 채점하기시간 설정 → OMR 마킹 → 정답 입력 순서입니다.

화면에서 하는 일

시간 설정

자료에 맞춰 전체·과목 시간을 바꿉니다.

답안 마킹

문제를 풀 때마다 OMR에 바로 표시합니다.

채점과 복기

정답을 입력하고 오답과 미응답을 확인합니다.

PC 화면과 팝업 사용을 권장합니다. 모바일에서도 열 수 있지만 여러 도구를 함께 쓰기에는 화면이 좁습니다.
