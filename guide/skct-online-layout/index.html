화면 배치

문제는 왼쪽에.
도구는 오른쪽에.

문제 자료 옆에 팝업을 두고 계산기, 메모장, 그림판을 짧게 오가는 흐름을 익혀보세요.

기본 동선

문제 확인 → 필요한 도구 사용 → 답 결정. 매 문항 같은 순서를 유지합니다.

도구는 필요할 때만

메모와 계산기를 계속 열어 두지 마세요. 필요한 순간에 열고 바로 닫습니다.

설정

자료에 맞춰 시간만 조정한 뒤 시작합니다. 보조 도구의 위치는 연습 중 바꾸지 않는 편이 좋습니다.

추천 루틴

  1. 5문항 동안 동선을 고정합니다.
  2. 10문항 동안 시간 상한을 적용합니다.
  3. 끝나면 미응답과 조작 지연만 확인합니다.
주의: 이 사이트는 공식 문제나 기출문제를 제공하지 않으며 SK그룹 또는 SKCT 공식 운영기관과 제휴된 서비스가 아닙니다. 실제 시험 화면, 영역 구성과 규정은 달라질 수 있으므로 공식 안내를 기준으로 확인하고, 저작권이 있는 문제 전문을 커뮤니티나 기록 공유 용도로 게시하지 마세요.
