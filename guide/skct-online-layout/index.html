화면 배치

문제는 왼쪽에.
도구는 오른쪽에.

문제 자료 옆에 팝업을 두고 계산기, 메모장, 그림판을 짧게 오가는 흐름을 익혀보세요.

기본 동선

문제 확인 → 필요한 도구 사용 → 답 결정. 매 문항 같은 순서를 유지합니다.

도구는 필요할 때만

메모와 계산기를 계속 열어 두지 마세요. 필요한 순간에 열고 바로 닫습니다.

설정

자료에 맞춰 시간만 조정한 뒤 시작합니다. 보조 도구의 위치는 연습 중 바꾸지 않는 편이 좋습니다.

추천 루틴

  1. 5문항 동안 동선을 고정합니다.
  2. 10문항 동안 시간 상한을 적용합니다.
  3. 끝나면 미응답과 조작 지연만 확인합니다.

실제로 배치하는 순서

1. 문제 자료를 먼저 고정합니다

PDF나 E-book을 왼쪽에 두고 문항 본문과 표가 한 번에 보이는 배율을 정합니다. 자료 창을 먼저 고정해야 팝업 너비를 바꿔도 문제 위치가 흔들리지 않습니다.

2. 도구 팝업을 남은 공간에 맞춥니다

오른쪽에 팝업을 두고 타이머, 현재 과목과 답안 버튼이 잘리지 않는지 봅니다. 계산기나 그림판은 필요한 순간에만 열어 문제 본문을 가리지 않게 합니다.

3. 5문항으로 동선을 시험합니다

문제 확인 → 필요한 숫자 표시 → 계산 또는 메모 → 답 결정 → 다음 문항 순서를 다섯 번 반복합니다. 창을 다시 찾거나 마우스 이동이 길어진 지점만 적습니다.

4. 한 요소만 바꿉니다

팝업 너비, PDF 배율과 도구 위치를 한꺼번에 바꾸면 무엇이 좋아졌는지 알 수 없습니다. 다음 5문항에서는 가장 불편했던 요소 하나만 조정합니다.

화면 문제가 기록에 섞이지 않게

팝업이 열리지 않음

브라우저의 팝업 차단 표시를 확인하고 이 사이트를 허용합니다. 새 창이 뒤에 숨어 있지 않은지 확인한 뒤 한 번만 다시 엽니다.

버튼이 잘리거나 가로로 넘침

브라우저 배율을 100%로 되돌리고 창을 조금 넓힙니다. 작은 화면에서는 자료와 도구를 동시에 보기보다 문제 확인과 기록을 순서대로 전환하는 편이 낫습니다.

타이머가 예상과 다름

연습 시작 전에 전체 시간과 과목 시간이 자료의 기준과 맞는지 확인합니다. 화면 문제로 중단했다면 해당 회차를 실력 기록과 섞지 말고 환경 점검 회차로 표시합니다.

완료 뒤 확인할 것

오답 수보다 조작 때문에 멈춘 횟수, 팝업을 다시 배치한 횟수와 미응답을 먼저 봅니다. 다음 회차의 화면 설정은 이 세 가지 중 하나만 줄이는 방향으로 정합니다.

주의: 이 사이트는 공식 문제나 기출문제를 제공하지 않으며 SK그룹 또는 SKCT 공식 운영기관과 제휴된 서비스가 아닙니다. 실제 시험 화면, 영역 구성과 규정은 달라질 수 있으므로 공식 안내를 기준으로 확인하고, 저작권이 있는 문제 전문을 커뮤니티나 기록 공유 용도로 게시하지 마세요.
