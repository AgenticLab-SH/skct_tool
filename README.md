# SKCT 연습 도구 공개 웹사이트

이 저장소는 [skct.agenticfabworks.com](https://skct.agenticfabworks.com/)에 공개되는 정적 파일만 관리합니다.

## 저장소 역할

- 승인된 HTML, CSS, JavaScript, 이미지 등 공개 산출물만 둡니다.
- 앱 원본, 백엔드, Firebase 운영 자료, 배포 런북, 감사 기록은 별도 비공개 저장소에서 관리합니다.
- `main` 반영 시 GitHub Pages가 `skct.agenticfabworks.com`으로 배포합니다.

## 공개 전 확인

1. 비공개 원본 저장소의 공개 내보내기 스크립트와 허용 목록을 사용합니다.
2. 개인정보, 자격 증명, 내부 문서, 백엔드 코드가 없는지 검사합니다.
3. 상대 경로, 주요 JavaScript, 모바일·데스크톱 화면을 확인합니다.
4. 승인 후에만 `main`과 운영 사이트를 갱신합니다.

공개 이슈에도 개인정보, 자격 증명, 보안상 민감한 내용을 남기지 마십시오.
