SKCT 화면에는 광고 없음

풀이 앱, 소개, 가이드, 가격과 기록 화면을 포함한 SKCT 하위 도메인에는 광고 로더를 두지 않습니다.

왜 이 도구를 만들었나요?

문제보다 화면 전환이 낯설었습니다

보유한 문제 자료를 보며 계산기, 메모와 시간을 따로 관리하면 창을 찾는 동작이 풀이 흐름을 끊었습니다. 그래서 문제 콘텐츠를 복제하지 않고 화면 도구와 기록만 한곳에 모았습니다.

한 번의 점수보다 반복 흐름을 봅니다

답안 기록은 실제 시험 OMR을 흉내 내기 위한 기능이 아닙니다. 같은 자료를 여러 번 풀 때 정답률, 미응답과 문항 시간을 비교하기 위한 개인 채점 도구입니다.

직접 확인하는 범위

브라우저와 화면

팝업 열기, 창 배치, 타이머 시작·일시정지, 계산기 입력, 메모장·그림판 전환과 모바일 세로 배치를 실제 브라우저에서 확인합니다.

기록과 내보내기

기본 채점, 문항 이동, 회차 저장, CSV 내보내기와 기간이 끝난 뒤의 제한 상태를 기능 흐름별로 확인합니다. 공개 가이드와 실제 버튼 이름이 어긋나면 함께 수정합니다.

확인하지 않는 것

공식 문제 유형, 합격선, 최신 시험 영역과 실제 응시 화면의 동일성을 검증하지 않습니다. 이 정보는 SK 채용과 공식 검사 안내가 유일한 기준입니다.

오류와 정정

문제가 발생한 URL, 브라우저, 화면 크기와 재현 순서를 이메일로 보내면 확인합니다. 서비스 전체의 검증·수정 절차는 콘텐츠 검증 원칙에 공개합니다.

SK그룹 또는 공식 SKCT 운영기관과 제휴하지 않은 개인 제작 연습 도구입니다. 문제·정답·합격을 제공하거나 보장하지 않습니다.
